# Read-Content-From-Google-Sheets

Projeto exemplo utilizando as API's do google e o google script para leitura de uma planilha.

O projeto possui 2 exemplos um sando table e outro usando div:

### [Exemplo com Table](https://github.com/duanbressan/Read-Content-From-Google-Sheets/tree/master/Example%20Using%20Tables)
Primeira implementação minha usando javascript e carregando os elementos em uma tabela.

link: http://read-google-sheets-table.bitballoon.com

### [Exemplo com Div](https://github.com/duanbressan/Read-Content-From-Google-Sheets/tree/master/Example%20Using%20Div)
O segundo projeto é uma melhoria do primeiro aonde os arquivos de css, javascrip e html foram separados e os elementos carregados em div's.
